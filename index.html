Youtube Translator

This application emphasizes the use of Javascript to generate a video player, that doesn't play ads, and also allows you to translate words. It contains an English to Spanish and Spanish to English translator where it allows for the user to write a word in the comment box and see the result.

Please enable Javascript to get the best use out of this application.

To get started, please select a country you'd like to look up, and click Search.
